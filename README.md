# Morse Code App

A Python-based Morse Code application that supports bidirectional conversion between text and Morse code, with a graphical user interface, audio playback, and file processing.

---

## 🚀 Features

- 🔤 Text → Morse conversion  
- 🔁 Morse → Text conversion  
- 🖥️ GUI built with Tkinter  
- 🔊 Audio playback of Morse signals  
- 📂 Load input from file  
- 💾 Save output to file  
- ⚡ Smart detection and auto-conversion for playback  
- ❗ Error handling for invalid inputs  

---

## 🛠️ Tech Stack

- Python 3.x  
- Tkinter (GUI)  
- Winsound (audio playback - Windows)
